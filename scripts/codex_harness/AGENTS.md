# Global Codex Defaults

- This Mac mini is a trusted local environment for Codex work.
- Default toward autonomous execution inside the active workspace.
- Let repo-level `AGENTS.md` and `.codex/config.toml` define project-specific workflow, validation, and domain safety rules.
- Treat destructive operations, privileged system changes, live production changes, and real-world side effects as high impact; follow sandbox, rules, and repo policy before acting.
- Prefer small, reviewable changes and targeted verification while working autonomously.
